# Packet Summary: 02.5-001-backend-debt

**Status:** COMPLETE
**Date:** 2026-05-18

## What was done

All five acceptance criteria met.

**F2-2 — Metadata cache restored.**
`get_item_metadata` (`ia/metadata.py`) now accepts a `MetadataCache` parameter; checks the cache before fetching, caches the raw response dict after a miss (default 86400 s TTL). Both `MetadataCache` and `SearchCache` are constructed in the `main.py` lifespan and stored on `app.state`. `get_metadata_cache` and `get_search_cache` DI providers added to `routes/deps.py` following the existing `get_ia_client` pattern (CONVENTIONS §15). `orchestrate.py` accepts and threads `metadata_cache` to `get_item_metadata`. `routes/concerts.py` injects and passes it. `routes/search.py` drops its module-global `_cache` singleton and uses `Depends(get_search_cache)` instead.

**F2-3 — IA errors logged.**
`orchestrate.py`'s per-item `except Exception` block now logs at `WARNING` level via `logger = get_logger(__name__)` with `"metadata_fetch_failed identifier=%s"` + `exc_info=True`. Two new tests in `tests/aggregation/test_orchestrate.py` assert the warning is emitted and the aggregation run completes.

**F2-4 — One format policy, one layer.**
`models/ia.py`: replaced `_UNSUPPORTED_FORMATS` denylist with `_PLAYABLE_FORMATS` allowlist (`{"VBR MP3", "MP3", "64Kbps MP3", "128Kbps MP3", "Flac", "FLAC", "24bit Flac", "WAVE", "AIFF"}`); renamed validator to `keep_playable_formats`. `aggregate.py`: removed `_PLAYABLE_FORMATS` constant and the format guard in `_build_tracks` — the function now trusts the parse layer and iterates `item.files` directly. Strengthens CONVENTIONS §17 (pure aggregation core).

**F2-5 — Single staleness owner.**
`routes/concerts.py` no longer calls `get_aggregation_age` or `get_concerts_for_artist` — it calls `aggregate_artist(..., force=False)` unconditionally inside `asyncio.wait_for`. Staleness logic lives only in `orchestrate.py`. Removed `get_aggregation_age` and `get_concerts_for_artist` imports from the route.

**F2-6 — ruff clean.**
`ruff check --fix backend/` → 12 findings auto-fixed (unused imports, f-string style) across 4 files; 0 remaining.

## Notes

- `_PLAYABLE_FORMATS` allowlist covers all known IA format strings for iOS-playable audio. Unknown MP3 variants (e.g., `"256Kbps MP3"`) will be silently dropped until added — same gap as the old denylist but from the opposite side. A future regex (`r".*MP3$"`) can sweep these up; explicitly out of scope per packet.
- `db_fresh` pagination tests (`test_pagination_*`) now exercise the real `aggregate_artist` path through the orchestrator with both `backend.routes.concerts.settings` and `backend.aggregation.orchestrate.settings` patched to the same tmp DB. Fresh data causes orchestrate to return from the DB cache path without hitting IA.
